Welcome to my website

This is a sample HTML page used to demonstrate HTML5 principles and techniques. We are using Express JS. Express is a node js web application framework that provides broad features for building web and mobile applications. It is used to build a single page, multipage, and hybrid web application. It's a layer built on the top of the Node js that helps manage servers and routes.

Site Contents

This site consists of 3 static views ( Home, About, HTML Demo) as well as 5 routes that supply data , including: All Students, Students by Course, A Single Student By Student Number, All Managers and All Courses

Site Information

This site utilizes HTML5: a markup language used for structuring and presenting content on the World Wide Web. HTML stands for Hyper Text Markup Language. HTML describes the structure of a Web page and contains different elemnts. These elements tells browser on how to display the contents.
